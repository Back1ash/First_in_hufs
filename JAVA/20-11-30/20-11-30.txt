1. java.lang 패키지에 대해 공부했다. 
 	- Object 개념을 이해하고 교과서에서 예제를 보며 이해했다.
	- 다양한 이미 선언된 클래스들을 살펴보며 하위 메소드까지 어떻게 사용하는지 학습했다.

2. java.util 패키지에 대해 공부했다.
	-Random 클래스에 대해 공부했다. 파이썬의 Random 라이브러리와 큰 차이가 없는 것 같아 이해하기 간편했다.
	-문자열 파싱에 대해 공부했다. 문자열 파싱의 경우 파이썬과 C언어에서도 다뤘던 개념이라서 자바에서의 문법과 세부 개념들만 이해하고 넘어가니 학습하기 쉬웠다.
	-Date 클래스에 대해 공부했다. 다만 Date 클래스의 경우 사용함에 있어서 메소드와 인자가 길어 두고두고 다시 봐야할 것 같다.
	-Canlender 클래스는 메소드 이름 자체부터 직관적이어서 바로 사용할 수 있을 것 같다. (*Month 변수만 -1상태인 것을 기억하면 될 것 같다.)

3. Collection 개념에 대해 공부했다.
	-Collection은 Set, List, Queue(인터페이스)로 이루어져있고 각 객체마다 자신을 이루고 있는 추상 클래스가 존재한다.
	-LinkedList | ArrayList, Vector는 순서를 정하는 방식에 따라 구분된다. LinkedList는 지목하는 커서가 존재하고 그 외는 흔한 배열처럼 한 막대기 안에 들어가있다고 생각하면 될 것 같다.
	-다만 ArrayList는 여러 스레드에서 동시에 사용하면 데이터가 망가지고 Vector는 데이터가 안전하다.
	-Iterator 인터페이스 사용 예를 보며 감을 잡아가는 중이다. 사용법은 "Iterator<자료타입> 변수명"이다.
